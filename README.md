## 使用 Webpack + Typescript 搭建 Phaser3 项目

### 准备工作

- 需要安装 [Node.js](https://nodejs.org) 。

### 下载本项目

`git clone https://github.com/YuQian2015/phaser3-typescript.git`

### 运行

`cd phaser3-typescript`

| 命令          | 描述         |
| ------------- | ------------ |
| `npm install` | 安装所需模块 |
| `npm start`   | 启动预览     |
